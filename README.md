# personal-library-app
Веб-приложение для управления личной библиотекой книг
---
# Функциональные требования (User Stories)

## 1. Добавление книги
**Как** пользователь библиотеки  
**Я хочу** добавлять новые книги с указанием названия, автора, года и статуса  
**Чтобы** вести учет своих книг

## 2. Поиск книг
**Как** пользователь  
**Я хочу** искать книги по названию или автору  
**Чтобы** быстро находить нужные книги

## 3. Просмотр списка
**Как** пользователь  
**Я хочу** видеть все книги в виде удобного списка  
**Чтобы** иметь общее представление о библиотеке

## 4. Изменение статуса
**Как** пользователь  
**Я хочу** менять статус книги (Планирую/Читаю/Прочитано)  
**Чтобы** отслеживать прогресс чтения

## 5. Редактирование книги
**Как** пользователь  
**Я хочу** редактировать информацию о книге  
**Чтобы** исправлять ошибки или обновлять данные

## 6. Удаление книги
**Как** пользователь  
**Я хочу** удалять книги из библиотеки  
**Чтобы** поддерживать актуальность списка
